基于Github的个人博客搭建指南

Posted on 2018-09-18 | In Mr , Github

随着信息网络社会的蓬勃发展，电子化的趋势越来越明显，博客、公众号、微博、短视频……都聚集着大量的用户，本文主要介绍如何使用Hexo基于GitHub搭建个性化的博客页面。

Hexo简介


这是hexo的中文官方网站，其定位是一个快速、简洁且高效博客框架。Node.js 所带来的超快生成速度，让上百个页面在几秒内瞬间完成渲染。Hexo 支持 GitHub Flavored Markdown 的所有功能，甚至可以整合 Octopress 的大多数插件。通过Hexo可以方便的生成静态网页托管在GitHub或者Heroku上，Hexo已经是搭建博客的主流框架选择。

GitHub Page

这是GitHub Page的官方文档，GIthub Pages可以放置网页文件到指定文件夹，然后提供一个专属域名用于展示一些项目，很适合用于静态博客页面的展示。
